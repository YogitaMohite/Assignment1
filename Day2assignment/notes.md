1. Function
A function is like a small machine.
When you call it, it does some work for you.

2. Anonymous Function (Function with no name)
This function has no name.
We store it inside a variable.

3. Arrow Function (⇒)
This is a short and modern way of writing a function.

4. Conditional Block (if-else)
It checks a condition.
If the condition is true, do one thing.
If false, do something else.

5. Callback Function
A function that is passed inside another function as an argument

6. Higher Order Function
takes another function as a parameter
OR
returns another function
is called a Higher Order Function.

7. What is DOM?
DOM is the structure of a webpage.
JavaScript uses DOM to read or change HTML.

8. DOM Manipulation
Changing HTML with JavaScript is called DOM manipulation.
Examples:
create element
add element
change text
change color

9. Event Listeners
Event listener means:
“When something happens, run this code.”
Example:
button click
mouse enter
mouse leave

10. Mouse Events
Mouse events are actions done by mouse.
Examples:
click
mouseenter (mouse goes inside)
mouseleave (mouse goes out)
