Java Exception Handling Notes

What is an Exception?

An exception is an unwanted error that stops the normal flow of a program.

Why Exception Handling?

  • To avoid program crash
  • To handle errors smoothly
  • To show custom user-friendly messages

Keywords

  • try → code that might cause error
  • catch → handles the error
  • finally → always runs (optional)

Simple Example

public class Demo { public static void main(String[] args) { try { int a = 10; int b = 0; int c = a / b; // risky line System.out.println(c); } catch (Exception e) { System.out.println("Error occurred: " + e.getMessage()); } } }

Output

Error occurred: / by zero
