JavaScript Notes

1. Difference between var, let and const

used to create variables in JavaScript.

  • var:Variable can be redeclared & Updated.A global scope variable.
  • let:Variable can not be redeclared but can be Updated.A local scope variable.
  • const:Variable can not be redeclared or Updated.A local scope variable
var name = "Yogita";
name = "Priya";

let age = 20;
age = 21;

const city = "Pune";
city = "Mumbai"; // Error yenar ithe

2. Hoisting

JavaScript moves variables and functions to the top before running.

console.log(x);
var x = 10; // Output: undefined
console.log(y);
let y = 20; // Error

3. Function

A function is a reusable block of code.

function add(a, b){
  return a + b;
}

add(5, 3); // 8

4. Static vs Dynamic Language

Static: Type is fixed (C, C++,Java).

int age = 20;

Dynamic: Type can change (JavaScript, Python).

let x = 10;
x = "Hello"; // allowed

6. Array and Object

Array: List of values.

let fruits = ["Apple", "Mango", "Banana"];
fruits[1]; // Mango

Object: Key-value data.

let student = {
  name: "Yogita",
  age: 20,
  city: "Pune"
};

student.name; // Yogita
